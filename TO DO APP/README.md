# 📝 To-Do List Web App

A simple and modern **To-Do List application** built using **HTML, CSS, and JavaScript**.  
This app allows users to add, delete, and manage daily tasks with automatic data saving using LocalStorage.

---

## 📌 Overview

The To-Do List App helps users organize their daily activities efficiently.  
Tasks can be added and removed dynamically, and all data is stored in the browser so tasks remain available even after refreshing the page.

This project demonstrates core frontend concepts such as **DOM manipulation**, **event handling**, and **local storage management**.

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Responsive UI)
- JavaScript (ES6)
- Browser LocalStorage

---

## ✨ Features

- ➕ Add new tasks
- 🗑 Delete tasks
- 💾 Persistent storage using LocalStorage
- ⌨️ Add task using Enter key
- 🎨 Modern and clean UI
- 📱 Responsive design
