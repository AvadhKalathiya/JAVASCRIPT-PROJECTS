# ⌨️ Typing Master Web App

A modern typing speed test application built using **HTML, CSS, and JavaScript**.  
This project helps users improve typing speed by measuring **WPM (Words Per Minute)**, **CPM (Characters Per Minute)**, and typing accuracy in real-time.

---

## 📌 Overview

Typing Master is a browser-based typing practice application that provides a clean UI and real-time performance tracking.  
Users type a given paragraph while the system calculates speed, mistakes, and remaining time dynamically.

This project focuses on **DOM manipulation**, **event handling**, and **real-time UI updates** using vanilla JavaScript.

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Modern UI Design)
- JavaScript (ES6)
- Local Development Server (npx serve)

---

## ✨ Features

- ⏱ Live typing timer
- ⚡ Real-time WPM calculation
- 🔥 CPM (Characters Per Minute)
- ❌ Mistake counter
- 🎯 Active character highlighting
- 🔄 Restart typing test
- 📱 Responsive design
- 🎨 Modern UI/UX layout
